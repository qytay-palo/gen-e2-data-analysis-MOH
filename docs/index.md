## üìÅ Project Structure

### Data Pipeline
- **data/raw/** - Original, immutable source data
- **data/interim/** - Intermediate transformation outputs
- **data/processed/** - Final cleaned datasets ready for analysis
- **data/external/** - External reference data (demographics, benchmarks)

### Analysis Workflows
- **notebooks/exploratory/** - Initial data exploration and EDA
- **notebooks/analysis/** - Final analysis notebooks with findings
- **src/** - Production-ready source code
  - `data_processing/` - ETL and data cleaning scripts
  - `visualization/` - Custom plotting functions
  - `analysis/` - Statistical analysis modules
  - `utils/` - Helper utilities

### Results & Reporting
- **results/tables/** - Summary statistics and analytical tables
- **results/metrics/** - Performance metrics and KPIs
- **results/exports/** - Stakeholder-ready data exports
- **reports/figures/** - Generated visualizations
- **reports/dashboards/** - Interactive dashboard applications
- **reports/presentations/** - Executive summaries and presentations
